# Java for-loop

![image](https://github.com/user-attachments/assets/51aac506-ca0c-499f-ad43-3e9542a4a0d7)

```java
public class Main {
  public static void main(String[] args) {
    int sum = 0;
    for (int i = 1; i <= 5; i++) {
      sum = sum + i;
    }
    System.out.println("Sum is " + sum);
  }
}
```

```java
public class Main
{
    public static void main(String[] args)
    {
        String[] cars={"Volvo","BMW","Benz"};
        for(String i:cars)
        {
            System.out.println(i);
        }
    }
}
```

output:

```
Volvo
BMW
Benz
```
