# Java while loop

![image](https://github.com/user-attachments/assets/4ed1da3e-3011-49b5-b789-1d821de4cdbf)

```java
public class Main 
{
  public static void main(String[] args) 
  {
    int i = 0;
    while (i < 5) 
    {
      System.out.println(i);
      i++;
    }  
  }
}
```

```java
public class Main 
{
  public static void main(String[] args) 
  {
    int i = 0;
    do 
    {
      System.out.println(i);
      i++;
    }
    while (i < 5);  
  }
}
```

output:
```
0
1
2
3
4
```
